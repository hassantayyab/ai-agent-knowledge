# Project Instructions (Claude Code)

This repo is an LLM-optimized knowledgebase built from two books:

- `principles/` = conceptual foundations (“what to build”)
- `patterns/` = pragmatic playbooks (“how to build”)

## Retrieval-first workflow (mandatory)

1. Read `INDEX.md`.
2. Select up to **2** relevant chapters.
3. Read them and answer using their concepts + code.
4. If still missing information, read **1** additional chapter max.

## Prefer

- Reusing code blocks from chapters
- Using `/skills/*.md` as quick recipes when applicable

## Do not

- Do not load all chapters.
- Do not answer from general knowledge if the repo contains the relevant info.

## Answer format

- Selected chapters:
- Why these:
- Answer:
- References: (chapter + section headings)
